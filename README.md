# Customer Churn Analysis using Python

## Project Overview

This project analyzes customer churn behavior in a telecom company using Python, Pandas and Matplotlib.

## Dataset Summary

- Records: 7,043
- Features: 21

## Data Cleaning

- Checked missing values
- Checked duplicates
- Converted TotalCharges to numeric format

## Key Findings

### Churn Rate
- 26.54% of customers churned.

### Contract Type
- Month-to-month customers showed the highest churn.
- Two-year contract customers showed the lowest churn.

### Monthly Charges
- Churned customers paid higher monthly charges on average.

### Customer Tenure
- Churned customers had significantly lower tenure.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Kaggle

## Business Insights

The analysis indicates that contract type, monthly charges, and customer tenure are strong factors influencing customer churn.
